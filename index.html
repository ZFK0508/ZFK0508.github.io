Project Idea

Overview:

This project is building a platform for children to programming and learn programming through making the code into icons. Icons can let children easier to understand what its function so that children can combine the codes like LEGO which can attract children’s interest. Children can check the code and its meaning by clicking the icons if they want to learn more and customize icons. Children can create games or programs and share on the platform. They can also share their experience about building games or programs on the platform. More than that, the platform can provide lessons for children about programming and some programming games.

Motivation:

Nowadays, people can’t live without smart devices and apps on them. Relatively, the demand of learning programming is growing. The world will need more and more developers. In Europe alone, there will be over 800,000 ICT job vacancies (Balanskat and Engelhard, 2015). Many parents start to let their children learn programming. Let children learn programming can help them make a good preparation for their careers or inspire their interest of programming. However, it’s boring when start learning programming. If make the codes become some icons or make programming like games can inspire children’s interest. Plass et al. (2015) states that “...it involves the use of game elements, such as incentive systems, to motivate players to engage in a task they otherwise would not find attractive.” Children can learn better if they have more interest in something. Also, communicate with others can also help children learn better.

Description:

First of all, users should register an account to login the platform. There is a module for users to programming. In the module, users can program via dragging and dropping icons form the bar on the left. Each icon has different meanings and code inside like for loop or some basic code. Users can check the code inside by double clicking the icon. Combining icons to build a whole program. In order to avoid some grammar mistakes, some icons can only combine with specific icons. Visual programming is easier for users to figure out what the logic of the program but it’s hard to learn how the codes achieve the function. To fix this, users can customize icons by written codes inside the icons to achieve specific function. Also, users can choose different programming languages inside the icons like Java, C/C++ or Python. However, this is more suitable for advanced students. Besides, users can convert the combinatorial icons into text-based languages.

The platform also provides many lessons and article for users to learn about programming. Users can upload articles or videos to their channels on the platform to teach other users. The content can be teaching children the logic of programming, some abstract conceptions, basic programming languages or some examples about how to achieve somethings. Video lesson can be free or paid, it depends on the creators’ choice. Meanwhile, all the articles are free to read but users can donate for the articles. Also, users can subscribe authors’ channel. There are three hot lists, one for videos, one for articles and the authors. Their ranking depends on algorithm basic on “likes”, views, reading, subscriptions and donations. The authors can get paid by creating lessons and it can encourage them to create better lessons and attract more users to create lessons. Of course, all then content of videos and articles must be checked by platform, non-compliant, illegal or not related content are not allowed to publish. The platform will warn the authors and require them to modify. If they don’t modify after few times of warning, their accounts will be banned. Users can also report the content of the videos or articles. The main clients of the platform are children so the platform should set a good example for children.

When users finish their programs or games, they can upload to the library on the platform. Users can also view others’ programs or games and download them. The games or programs also can be viewed in users’ channels. Users can not only donate for others’ lessons, also the programs or games they written. They can leave comments and communicate with others under the pages of games or programs. Adding friends is also a good choice. Communication and share experience can help users learn programming better. But all these above need to follow the community rules otherwise users will be banned. Besides, the platform will list the most popular games and programs once a season. The author of the games or programs will get prize.

For young children, they can program via dragging and dropping icons to create programs which can highly attract their interest just like playing games. For advanced students, they can check the basic code of icons and learn further about programming.
